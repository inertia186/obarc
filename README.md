# OpenBazaar API Ruby Client

A simple OpenBazaar API client for Ruby.

### Installation

Add the gem to your Gemfile:

    gem 'obarc', github: 'inertia186/obarc'
    
Then:

    $ bundle install

### Usage

```ruby
require 'obarc'

session = OBarc::Session.new(username: 'username', password: 'password')
```
